# Htstr 1UP

Single player Hitster game - place songs on a timeline by guessing their release year.

**Play now:** https://djaenecke.github.io/htstr-1up/

## How to Play

1. Login with your Spotify Premium account
2. Configure game settings:
   - **Easy Mode**: Compare to the last placed card (before/after)
   - **Expert Mode**: Insert card into correct position in timeline
   - **Timed Mode**: Must place before song ends, no pause/replay
3. Listen to the song and tap a drop zone to place the card
4. Card flips to reveal the year - correct placements add to your score

## Features

- Two difficulty modes (Easy / Expert)
- Optional Timed Mode for extra challenge
- Configurable playback duration (15/30/45/60 seconds or full song)
- Random or fixed start position
- Win conditions: 5/10/15/20 correct placements or unlimited
- 17 Hitster editions included
- Card colors matching the tabletop game
- Works on mobile and desktop

## Requirements

- Spotify Premium account (required for Web Playback SDK)
- Modern browser with JavaScript enabled
